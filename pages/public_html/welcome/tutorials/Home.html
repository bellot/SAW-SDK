What is SAW SDK ?

SAW SDK is a Software Development Kit that allows Java developpers to easily build Web applications. SAW SDK is not a Content Management System (CMS) nor a Web Applications Sever (WAS). SAW SDK allows you to develop an entire Web application that acts as a server. SAW SDK requires Java programming.

Thanks to the functions proposed by SAW SDK, the programming effort is reduced to the minimum, mainly to program the Business Logic of the application. These functions propose mechanisms that makes the Web Application much more efficient than equivalent applications developped in other frameworks.

Proof of concept. SAW SDK has been used to develop a fully functional and complete Web application for Conference Management including all aspects of a conference. It can be seen here.
