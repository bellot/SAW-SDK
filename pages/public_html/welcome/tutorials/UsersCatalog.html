The Users Catalog

The Users Catalog is the data structure containing the information on the users of the Web application. As explained in The User data structure, the user information is an object of the class User containg the user id, the login name, the encripted password and the privileges field of the user.

The Users Catalog is an object of class UsersCatalog. It is an Entity Binz: there exist a unique instance of it which is loaded from its workspace file the first time it is accessed and saved back to its workspace file when the server shut down. It is ket in memory during this interval. The Users Catalog can be accessed as a static variable of the class UsersCatalog: 

UsersCatalog.entity

By looking at the class UsersCatalog, we can see that this class allows to create a user, to retrieve a user using its user id or its login name, etc.
